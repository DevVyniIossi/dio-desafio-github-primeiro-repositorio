Módulo especifico para a explicação de como o Git funciona por "baixo dos panos".

No módulo o professor aborda sobre:

- SHA 1 : Secure Hash Algorithm, é um conjunto de funções hash criptográficas projetadas pela NSA (Agência de Segurança Nacional dos EUA).
- Objetos fundamentais, explica que os objetos são BLOBS, TREES e COMMITS.
- Sistema distribuido Seguro, explica que as versões do código em desenvolvimento, ficam salvo tanto na nuvem do Git e na máquina local.
